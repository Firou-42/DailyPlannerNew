# Changelog

Все заметные изменения в этом проекте будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2026-04-23

### Добавлено
- Главный экран с отображением задач на выбранную дату.
- Создание, редактирование и удаление задач.
- Календарь с недельной сеткой и переключением дат.
- Аналитика с графиком продуктивности и статистикой по категориям.
- Экспорт отчётов в форматы Microsoft Word (.docx) и Microsoft Excel (.xlsx).
- Настройки: тёмная и светлая тема, уведомления, первый день недели.
- Поиск задач по названию и описанию.
- Фильтрация задач по категориям.
- Повторяющиеся задачи (ежедневно, еженедельно, ежемесячно).
- Разбиение длинных задач на части.
- Проверка конфликтов времени.
- Выделение просроченных задач.
- Окно «О программе» с информацией о версии и разработчике.
- Автоматическое сохранение данных в SQLite.
- Сборка APK для Android через flet build apk.

### Исправлено
- Исправлена ошибка сохранения настроек без изменений.
- Исправлено создание пустого отчёта без предупреждения.
- Исправлена проверка времени окончания задачи.

### Изменено
- Название вкладки «Сегодня» изменено на «Главная».
- Панель навигации перемещена вниз для удобства мобильного использования.
